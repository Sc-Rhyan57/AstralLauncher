Place BepInEx Android build files here before building the APK.
Required structure inside assets/BepInEx/:
  core/    - BepInEx core DLLs (libil2cpp.so patcher, etc.)
  plugins/ - (empty — mods are added at patch time)

Download from: https://builds.bepinex.dev/projects/bepinex_be
Select: BepInEx Unity IL2CPP for Android (arm64-v8a)
